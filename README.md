# 个人网站项目

## 项目简介

这是一个现代化的个人网站项目，用于展示个人信息、作品集、技能和联系方式。

## 个人网站基础知识

### 1. 什么是个人网站？

个人网站是一个展示个人品牌、作品集、技能和经历的在线平台。它可以帮助你：
- 建立个人品牌形象
- 展示你的作品和项目
- 提供联系方式
- 分享个人博客或文章
- 提升职业形象

### 2. 个人网站的核心组成部分

#### 2.1 首页（Home）
- 个人简介
- 欢迎信息
- 导航菜单

#### 2.2 关于我（About）
- 个人背景
- 教育经历
- 职业经历
- 技能介绍

#### 2.3 作品集（Portfolio）
- 项目展示
- 作品分类
- 项目详情

#### 2.4 博客（Blog）
- 文章列表
- 文章详情
- 分类和标签

#### 2.5 联系方式（Contact）
- 联系表单
- 社交媒体链接
- 邮箱地址

### 3. 技术栈选择

#### 3.1 前端技术
- **HTML5**: 网页结构
- **CSS3**: 样式设计（可以使用 CSS 框架如 Tailwind CSS, Bootstrap）
- **JavaScript**: 交互功能（可以使用 React, Vue, 或原生 JS）
- **响应式设计**: 适配移动端和桌面端

#### 3.2 后端技术（可选）
- **静态网站**: 使用 Jekyll, Hugo, Next.js 等静态站点生成器
- **动态网站**: 使用 Node.js, Python Flask/Django, PHP 等
- **无服务器**: 使用 Vercel, Netlify 等平台

#### 3.3 部署平台
- **GitHub Pages**: 免费静态网站托管
- **Vercel**: 快速部署，支持多种框架
- **Netlify**: 静态网站托管和 CI/CD
- **Cloudflare Pages**: 全球 CDN 加速

### 4. 设计原则

#### 4.1 用户体验（UX）
- 简洁清晰的导航
- 快速加载速度
- 易于阅读的内容
- 直观的交互设计

#### 4.2 视觉设计（UI）
- 一致的配色方案
- 合适的字体和字号
- 充足的留白空间
- 高质量的图片和图标

#### 4.3 响应式设计
- 移动优先（Mobile First）
- 适配不同屏幕尺寸
- 触摸友好的交互元素

### 5. 内容规划

#### 5.1 个人简介
- 你是谁？
- 你做什么？
- 你的价值主张是什么？

#### 5.2 作品展示
- 选择最佳作品
- 提供项目描述
- 展示技术栈
- 添加项目链接

#### 5.3 技能标签
- 编程语言
- 框架和工具
- 软技能

### 6. SEO 优化

- 使用语义化 HTML
- 添加 meta 标签
- 优化图片（alt 属性）
- 创建 sitemap.xml
- 使用结构化数据

### 7. 性能优化

- 压缩图片
- 使用 CDN
- 代码压缩和合并
- 懒加载图片
- 缓存策略

## 项目结构

```
personal-website/
├── index.html              # About me（首页）
├── strategy-dashboard.html # Strategy Dashboard（预留：金融指标、策略表现）
├── notes-blogs.html        # Notes & Blogs（博客 + 笔记列表，预留 Notion 对接）
├── css/                    # 样式文件
│   └── style.css
├── js/                     # JavaScript 文件
│   └── main.js
├── images/                 # 图片资源
└── assets/                 # 其他资源
```

## 开始使用

1. 克隆或下载项目
2. 修改个人信息和内容
3. 自定义样式和设计
4. 部署到托管平台
